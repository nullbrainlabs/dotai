## Prerequisites

- Node.js 18 or later
- npm, pnpm, or yarn

## Install Globally

```bash
npm install -g @nullbrain/dotai
```

## Run Without Installing

Use `npx` to run dotai without a global install:

```bash
npx dotai init
npx dotai sync
```

## Verify

```bash
dotai --version
```
