## Why .ai?

<CardGrid>
  <Card title="Single source of truth" icon="document">
    Define rules, skills, agents, MCP servers, hooks, and permissions in one `.ai/` directory. No more scattered config files.
  </Card>
  <Card title="Multi-tool output" icon="puzzle">
    One `dotai sync` generates correct files for Claude Code, Cursor, Codex, and GitHub Copilot.
  </Card>
  <Card title="Conflict-aware" icon="warning">
    Content hashing detects when generated files have been manually edited. Never accidentally overwrite your changes.
  </Card>
  <Card title="Import existing" icon="open-book">
    Already have configs? `dotai import` scans your project and converts Claude, Cursor, Codex, or Copilot configs into `.ai/` format.
  </Card>
</CardGrid>
